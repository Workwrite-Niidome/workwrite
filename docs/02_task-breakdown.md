## 凡例

| 記号 | 意味 |
|------|------|
| 要件ID | 要件定義書の機能ID（R-xxx, A-xxx, S-xxx, E-xxx, RF-xxx）との対応 |
| 優先度 | P0=必須(MVP), P1=高, P2=中 |
| 担当 | FE=フロントエンド, BE=バックエンド, AI=AI/ML, INFRA=インフラ, DESIGN=デザイン |
| 依存 | 先行タスクのWBS ID |
| 工数 | 人日（1人あたり） |
