## 画面遷移の主要フロー

### 新規ユーザーフロー
```
登録 → オンボーディング診断 → トップページ（初回レコメンド表示）
```

### 読書フロー
```
トップページ/検索 → 作品詳細 → リーダービューア → 読後感タグ入力 → レビュー(任意) → 「次の自分」ナビ
```

### 作家投稿フロー
```
作家ダッシュボード → 新規作品作成 → エピソード投稿 → （AIスコア計算待ち） → スコア確認
```
