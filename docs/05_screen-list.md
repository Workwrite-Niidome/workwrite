## 画面遷移の主要フロー

### 新規ユーザーフロー
```
登録 → オンボーディング診断 → トップページ（初回レコメンド表示）
```

### 読書フロー
```
トップページ/検索 → 作品詳細 → リーダービューア → 余韻ページ → 感情タグ入力 → 状態変化スライダー → 「次の一冊」3作品 → レビュー(任意) → 本棚自動読了
```

### 作家投稿フロー
```
作家ダッシュボード → 新規作品作成 → エピソード投稿 → （AIスコア計算待ち） → スコア確認
```
