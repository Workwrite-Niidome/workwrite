# お知らせ

## 一部ページにおけるアクセス制御の不備について（修正済み）

いつもWorkwriteをご利用いただきありがとうございます。

一部のページにおいて、他のユーザーの作品編集画面や分析画面が表示される不具合がありました。現在は修正が完了しております。

### 影響範囲

- 作品編集ページおよびダッシュボードの作品分析ページにおいて、URLを直接入力した場合に他のユーザーの画面が表示される状態でした
- **作品データの変更（編集・削除）は全てサーバー側で著者本人のみに制限されており、他のユーザーによる作品の改変は一切行われておりません**

### 対応内容

- 全てのAPIエンドポイントにおける認可チェックの強化
- フロントエンド画面でのアクセス制御の追加
- 類似の問題がないか全エンドポイントの監査を実施

本件により、ご利用の皆様にご不安をおかけしましたことをお詫び申し上げます。今後このような問題が発生しないよう、セキュリティ体制を強化してまいります。
