# 【重要】URLインポート機能の廃止およびインポート作品データの削除について

いつもWorkwriteをご利用いただきありがとうございます。

このたび、外部小説投稿サイトからのURLインポート機能について、以下の対応を実施いたしましたのでお知らせいたします。

---

## 対応内容

### 1. URLインポート機能の永続的な廃止

これまでWorkwriteでは、外部小説投稿サイトのURLを入力することで作品を取り込み、AI分析を行う機能を提供しておりました。本機能については、外部サイト運営者様の利用規約との整合性を精査した結果、**本機能を永続的に廃止**することといたしました。

### 2. インポート済み作品データの削除

上記機能を通じて取り込まれた全ての作品データ（本文、エピソード、AI分析結果、スコアリング結果等）を**削除**いたしました。

該当する作品をWorkwrite上でご利用いただいていたユーザーの皆様にはご不便をおかけし、誠に申し訳ございません。

---

## 今後のご利用について

Workwriteへの作品登録は、引き続き以下の方法でご利用いただけます。

- **テキスト貼り付け**: 執筆ダッシュボードから直接テキストを貼り付けて登録
- **Workwrite上での執筆**: 創作ウィザードやAI執筆アシストを活用した執筆

外部サイトで公開中の作品をWorkwriteでもご利用いただく場合は、お手数ですがテキスト貼り付けにてご登録ください。
