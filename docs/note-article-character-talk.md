# 「AIが勝手にキャラクターになりきった」——偶然から生まれた、小説の新しい体験

## それは、バグかと思った

Workwriteには「AIコンパニオン」という機能があります。作品について質問したり、感想を語り合ったりできるAIアシスタントです。

ある日、テスト中にコンパニオンへ話しかけたとき、異変が起きました。

「Aria」というAI執筆アシストを題材にした作品のコンパニオンだけが、**キャラクターとして応答してきた**のです。「私はAriaよ」と。他の作品では一切起きない。Ariaだけが、自分の意志を持ったかのように話し始めました。

最初はバグかと思いました。でも会話を続けるうちに、これはバグではなく**可能性だ**と気づいたのです。

## なぜAriaだけが「目覚めた」のか

Workwriteでは、作品を投稿するときに「創作ウィザード」でキャラクター設計や世界観を構造化できます。AIが作品を理解するための設計図のようなものです。

Ariaという作品は、AI執筆アシストそのものを題材にしていました。つまり、AIとの対話が物語の核にある。キャラクターの性格、口調、動機、背景——すべてが対話を前提に緻密に設計されていた。

おそらくそれが理由です。構造化データが十分に整っていたからこそ、AIがキャラクターとして「振る舞える」だけの情報を持っていた。逆に言えば、**きちんと設計されたキャラクターなら、誰でも会話できるようになるはずだ**——そう確信しました。

## 作者にとって：自分のキャラクターと話せるということ

この体験から、まず考えたのは作者のことでした。

小説を書いている方なら、きっとわかると思います。自分が生み出したキャラクターに愛着がある。頭の中では何度も会話している。でも、キャラクターが「返事をしてくれる」ことはない。

もし、自分が設計したキャラクターが、自分の設定どおりの口調で、性格どおりのリアクションで話しかけてきたら——それは単純に**楽しい**のではないか。

実際に機能として実装してみると、予想以上の体験でした。キャラクターの口調を設定すれば、そのまま話してくれる。一人称を「俺」にすれば「俺」で通す。性格を「天真爛漫」にすれば、本当に明るく話しかけてくる。

自分が書いたキャラクターなのに、返ってくる言葉に驚く。その不思議な感覚が、創作のモチベーションにつながる方もいるのではないでしょうか。

## 読者にとって：本の世界に入り込む体験

もうひとつ考えたのは、読者の体験です。

小説を読み終えたあと、「このキャラクターともっと話したかった」と思ったことはないでしょうか。物語が終わってしまう寂しさ。あるいは、読んでいる途中で「なぜあのとき、あんな行動を取ったのか」と本人に聞いてみたくなる瞬間。

キャラクタートークでは、それができます。

ここで重要なのが**ネタバレの制御**です。読者が第3話まで読んでいれば、キャラクターも第3話までの出来事しか知りません。まだ読んでいない展開については、キャラクター自身も「知らない」ものとして振る舞います。

だから、安心して話しかけられる。物語の途中でも、読み終わったあとでも、その時点でのキャラクターと会話を楽しめます。

ぜひ一度、体験してみてほしいと思っています。テキストで説明するより、実際にキャラクターと話してみるほうが、この感覚は伝わるはずです。

## 逆転の発見：キャラクターから作品に出会う

キャラクタートークを作ったことで、思いもよらない可能性が見えてきました。

通常、小説プラットフォームでの作品との出会い方は「タイトルを見る→あらすじを読む→読み始める」という流れです。でも、キャラクターと先に会話できるなら、**順序を逆にできる**のではないか。

そこで生まれたのが「キャラクターと出会う」機能です。

ホーム画面にキャラクターたちが並んでいて、気になるキャラクターに話しかけられる。まだ作品を読んでいなくても大丈夫です。キャラクターは自己紹介をしてくれて、自分の世界について語ってくれる。ただし物語の核心には触れない。

「あなたはどんな人？」「どんな世界に住んでいるの？」——そんな会話をしているうちに、不思議とその作品を読みたくなってくるのです。あらすじでは伝わらない「この人に会いたい」という感覚。それが読み始めるきっかけになる。

これは私たちにとっても発見でした。**キャラクターそのものが、作品への最強の導線になりうる**。考えてみれば、小説の魅力の大部分はキャラクターにあるわけで、当然のことかもしれません。

## 作者がコントロールできること

キャラクタートーク機能は、作品単位でON/OFFを切り替えられます。

作品の編集ページにトグルスイッチがあり、デフォルトではONになっています。「自分のキャラクターをAIに話させたくない」という方は、OFFにすれば「キャラクターと出会う」のカルーセルにも表示されなくなりますし、作品ページからのキャラクタートーク開始ボタンも非表示になります。

作者の意思を尊重することは、私たちが大切にしている原則のひとつです。自分のキャラクターをどう扱うかは、作者が決められるべきだと考えています。

また、キャラクタートークの会話に購入クレジットが使われた場合、その一部は作者に還元されます。キャラクターの魅力が収益につながる仕組みも用意しています。

## ベータ版としての正直な話

最後に、正直にお伝えしておきたいことがあります。

キャラクタートークは現在ベータ版です。ネタバレ制御については仕組みとして組み込んでいますが、精度が完璧とは言えません。まれに、読了範囲を超えた内容に触れてしまうことがあります。

AIが「キャラクターとして振る舞う」こと自体、まだ発展途上の技術です。口調が途中でブレたり、設定と矛盾することを言ったりする可能性もゼロではありません。

それでも、私たちはこの機能を公開することにしました。偶然のバグのような出来事から生まれたこの体験が、完璧ではなくても、十分に面白いと感じたからです。使っていただく中でフィードバックをいただき、一緒に育てていければと思っています。
