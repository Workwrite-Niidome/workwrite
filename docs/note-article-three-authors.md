# 「こんな作品に出会いたかった」— 3人の作家と、その物語を紹介させてください

## はじめに

今日は、宣伝ではなく紹介をさせてください。

私たちが運営するWeb小説プラットフォーム「Workwrite」に、素晴らしい作品を書いてくださる作家さんたちが集まり始めています。読んで、震えました。こんな作品がWeb小説として生まれていることを、もっと多くの人に知ってほしい。

この記事では、3人の作家さんとその作品を紹介します。どれも無料で読めます。気になった作品があれば、ぜひ読んでみてください。

---

## 白縫いさやさん『紅薔薇』— 輪廻の果てで、あなたにもう一度会うために

**ジャンル**: SF / 文芸 / ロマンス
**全5話**

> 輪廻を超えて出会い続ける二人が、宇宙の終焉による永遠の別離に抗い、科学と物語それぞれの方法で再会の保証を求め続ける物語。

何度生まれ変わっても出会い続ける二人。その設定だけなら、よくあるロマンスかもしれません。しかしこの作品が描くのは、「宇宙が終わったあと」の話です。

輪廻が途絶える。もう二度と会えなくなる。その絶望に対して、一人は科学で、一人は物語で抗おうとする。SFとしてのスケールの大きさと、二人の間に流れる切実な感情の密度。5話という短さの中に、途方もない時間と愛が圧縮されています。

短い物語が好きな方、SFとロマンスの両方が好きな方に読んでほしい作品です。

[『紅薔薇』を読む → workwrite.jp](https://workwrite.jp/works/cmn37h0dr02jjr101qy85tc4c)

## YoShiKaさん『金の魔女と銀の娘』— 復讐と真実を追う、王道ファンタジーの本気

**ジャンル**: ダークファンタジー / 恋愛 / 冒険
**全123話**

> 金の髪と瞳を持つ魔女に故郷を滅ぼされた青年、テオドール。復讐の旅に出た彼が出会ったのは、魔女と瓜二つの顔をした少女、シェリア。銀の髪と瞳を持つシェリアは、魔女なのか。それとも。

123話。この物量が、まず凄い。

復讐のために魔女を追う青年と、魔女と同じ顔を持つ少女。「剣と魔法のファンタジー」というジャンルの王道ど真ん中を、圧倒的な筆力で駆け抜ける作品です。

テオドールの復讐心とシェリアへの感情の揺れ。一途で、切なくて、でも物語はどこまでもダークに進んでいく。王道だからこそ、書き手の力量が問われる。YoShiKaさんはそれに真正面から応えています。

長編ファンタジーに没頭したい方、ダークな世界観と一途な恋愛の組み合わせが好きな方に。プロローグの冒頭だけでも読んでみてください。引き込まれます。

> 「人も動物も関係ないわ。だって、あなた達、みんな死んじゃうでしょ？」

[『金の魔女と銀の娘』を読む → workwrite.jp](https://workwrite.jp/works/cmn6gq8cu00x7p4015klzlll3)

## 白蛇さん『銀世界の天球儀——解けない星座の観測記録』— 言葉の質量で殴られる純文学

**ジャンル**: 現代 / 純文学 / 百合 / 耽美
**全4話**

> 受験という無機質な座標軸に、私の魂は摩耗していた。逃げ込んだ最上階の図書室、そこは凍りついた琥珀の聖域。

> 「暗ければ暗いほど、星は正しく見えるものよ」

一文一文が重い。美しいのに、痛い。

受験に追われる主人公が逃げ込んだ図書室で出会う、月のように冷徹な司書課程の実習生。猛吹雪の夜、手の甲に刻まれたおまじない。合格という「救済」が、同時にこの静謐な場所からの「追放」を意味する——。

白蛇さんの文章は、Web小説の枠を超えています。「耽美」というタグが付いていますが、それは単なる装飾ではなく、この作品にはこの言葉でしか描けない世界がある。文章そのものが体験になる、稀有な書き手です。

純文学が好きな方、言葉の美しさに浸りたい方に、強くおすすめします。

[『銀世界の天球儀』を読む → workwrite.jp](https://workwrite.jp/works/cmn38kgqh04okr101joz21y87)

## こういう作品に出会える場所を作りたかった

3作品を紹介させていただきました。

SFロマンス、王道ダークファンタジー、耽美純文学。ジャンルはバラバラです。でも共通しているのは、**書き手が本気で書いている**ということ。そして、こういう作品がWeb小説の世界に確かに存在しているということ。

私たちがWorkwriteを作ったのは、こういう作品と読者が出会える場所が必要だと思ったからです。

更新頻度が高い作品がランキング上位に来る仕組みでは、5話の短編も、4話の純文学も、なかなか人の目に触れません。でも、作品の質を見れば、これらは間違いなく「読まれるべき物語」です。

Workwriteでは、AIが作品の質を6つの軸で評価し、感情タグで「どんな気持ちになれる作品か」を伝え、読者が自分の感性に合った物語と出会えるようにしています。更新頻度ではなく、作品そのものの力で、読者に届く。

もし今日紹介した3作品のどれかが気になったら、ぜひ読んでみてください。そして気に入ったら、作家さんにギフトレターで感想を届けてあげてください。120円から、あなたの「面白かった」が作家さんの収入になります。

まだ知らない物語が、あなたを待っています。

[Workwriteで作品を探す → workwrite.jp](https://workwrite.jp)
