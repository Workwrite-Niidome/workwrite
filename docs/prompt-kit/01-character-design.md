# プロンプト1: キャラクター設計

以下をChatGPT / Claude / Geminiにコピペしてください。
「」内のテーマ部分を自分の作品に合わせて書き換えてください。

---

```
あなたは小説の創作支援AIです。以下のテーマに合ったキャラクターを3〜5人設計してください。

【テーマ】
「ここにあなたの作品のテーマや構想を書いてください」

【ジャンル】
（例: ファンタジー、恋愛、SF、ミステリーなど）

【絶対遵守ルール】
- 各キャラクターの性別・一人称・口調・性格は設定内で必ず一貫させること
- 一人称は日本語の小説で自然なもの（僕/私/俺/あたし/わたくし等）を選ぶこと
- 口調の例文を必ず含めること（そのキャラクターらしいセリフを2つ以上）
- キャラクター同士の関係性を明確にすること

以下の形式で出力してください:

■ キャラクター名（役割）
  性別:
  年齢:
  一人称:
  性格: （2〜3文で詳しく）
  口調・話し方: （特徴 + セリフ例を2つ以上）
  外見:
  背景:
  動機・目的:
  他キャラとの関係性:
  成長アーク: （物語を通じてどう変化するか）
```

---

## 出力結果の使い方

出力されたキャラクター設定は、**この後のすべてのプロンプトに毎回含めてください**。
これがないと、AIはキャラクターの口調や設定を忘れます。
