# プロンプト2: プロット構築

前のステップで作ったキャラクター設定を「キャラクター設定」の部分に貼り付けてください。

---

```
あなたは小説のプロット設計の専門家です。
以下のキャラクターとテーマに基づいて、三幕構成のプロットを設計してください。

【テーマ】
「ここにあなたの作品のテーマを書いてください」

【伝えたいメッセージ】
（任意: 読者にどんなことを感じてほしいか）

【キャラクター設定】
（↓ プロンプト1の出力結果をここに貼り付け）

【絶対遵守ルール】
- キャラクターの性格・動機に基づいた展開にすること
- キャラクターの行動は性格設定から自然に導かれるものにすること
- 設定と矛盾する展開を提案しないこと
- 伏線を最低2つ計画すること（どの話で設置し、どの話で回収するか明示）

以下の形式で出力してください:

■ 前提: （物語の出発点を1〜2文で）

■ 中心的な葛藤: （主人公が直面する根本的な問い/対立）

■ テーマ: （この物語が探求するテーマを2〜3個）

■ 第一幕（序章）
  概要:
  主要イベント:
  感情の流れ:

■ 第二幕（展開）
  概要:
  主要イベント:
  転換点:
  感情の流れ:

■ 第三幕（結末）
  概要:
  主要イベント:
  クライマックス:
  感情の流れ:

■ 伏線計画
  1. （内容 / 設置: 第○話 / 回収: 第○話）
  2. （内容 / 設置: 第○話 / 回収: 第○話）
```
