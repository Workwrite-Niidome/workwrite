# プロンプト3: 章立て設計

プロンプト1（キャラクター）と2（プロット）の出力結果を貼り付けてください。

---

```
あなたは小説の構成設計の専門家です。
以下のキャラクター設定とプロットに基づいて、各話の詳細な章立てを設計してください。

【キャラクター設定】
（プロンプト1の出力を貼り付け）

【プロット構成】
（プロンプト2の出力を貼り付け）

【話数】
（例: 全5話）

【1話あたりの目安文字数】
（例: 2000〜3000字）

【絶対遵守ルール】
- 各話に登場するキャラクターを明示すること
- 各話の感情目標（読者にどんな感情を届けたいか）と強度（1-10）を設定すること
- 伏線の設置・回収がどの話で行われるかを明示すること
- 前の話の終わり方と次の話の始まり方に因果関係を持たせること

以下の形式で各話を設計してください:

■ 第1話「タイトル案」
  この話の概要: （2〜3文）
  主要シーン: （箇条書きで2〜3個）
  登場キャラクター:
  感情目標: （感情名）（強度: ○/10）
  伏線操作: （設置: ○○ / 回収: ○○ / なし）
  この話の終わり方: （次の話への引きをどう作るか）

■ 第2話「タイトル案」
  ...（同じ形式で全話分）
```

---

## ポイント

「この話の終わり方」を明確にしておくことが最も重要です。
次のステップで各話を書くとき、前話の終わりから自然に繋げるために使います。
