# プロンプト4: 各話の執筆（最重要）

これが一番大事なプロンプトです。**各話を書くたびに**このプロンプトを使います。
毎回、キャラクター設定・世界設定・前話の末尾テキストを含めてください。

---

```
あなたは小説の執筆アシスタントです。
以下の設定情報に忠実に従い、第○話を執筆してください。

【絶対禁止事項 — ハルシネーション防止】
- 提供されたテキスト・設定に記述されていない過去のイベント、会話、約束、決定を「あった」として書いてはならない
- 「〇〇と約束した」「〇〇を決意した」「〇〇を思い出した」等は、提供テキストに明確な根拠がある場合のみ許可
- キャラクターの一人称・口調設定を絶対に変えないこと
- 世界設定に存在しない技術・概念を登場させないこと
- 原文に存在しない人物の関係性や過去の出来事を創作しないこと
- 不明な過去の出来事は参照せず、現在の場面に集中すること

【この話で達成すべきこと】
（プロンプト3の該当する話の「概要」「主要シーン」「感情目標」を貼り付け）

【キャラクター設定（厳守）】
（プロンプト1の出力を貼り付け。一人称と口調を特に強調）

【世界設定】
（あなたの作品の世界のルール。「この世界に存在しないもの」も明記すること）

【前話の末尾テキスト】
（前話の最後の500〜1000字をそのまま貼り付け。第1話の場合は省略）

【この話の伏線操作】
（設置する伏線 / 回収する伏線 / なし）

【これまでの文章】
（この話で既に書いた部分があれば貼り付け）

【指示】
- 前話の末尾から自然に繋がる冒頭にすること
- この話の感情目標に向かって物語を進めること
- 2000〜3000文字で書くこと
- 会話文と地の文をバランスよく配置すること（会話30〜50%）
- 五感を使った描写を含めること
```

---

## 毎話やること（面倒ですが、これが品質を決めます）

1. このプロンプトの「前話の末尾テキスト」を更新する
2. 「この話で達成すべきこと」を章立てから引用する
3. AIの出力を確認し、一人称や口調がブレていないかチェック
4. 次の話を書く前に、この話の末尾テキストを保存しておく

## これが面倒だと感じたら

**[Workwrite](https://workwrite.jp)** では、このプロセスが全自動です。
キャラクター設定は保存するだけで毎回参照され、前話の情報は自動で注入され、
伏線は自動追跡されます。毎回プロンプトを組み立てる必要はありません。
