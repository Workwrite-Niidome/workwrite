## 3. クレジット消費の仕組み

### 3.1 消費順序（重要）

```
月次cr（毎月失効） → 報酬cr（30日で失効） → 有償cr（失効なし）
```

**作家に収益が発生するのは有償crが消費された場合のみ。**
月次crや報酬crでの利用では作家収益は0。

### 3.2 機能別クレジットコスト

| 機能 | モデル | コスト | 動的計算 |
|------|--------|--------|---------|
| コンパニオン（読書AI） | - | 0cr | 固定 |
| 校正・スコアリング・要約等 | Haiku | 0cr | 固定 |
| 文章アシスト（通常） | Sonnet | 1cr〜 | 動的（文字数依存） |
| 文章アシスト（思考モード） | Sonnet+思考 | 2cr〜 | 動的（文字数依存） |
| 文章アシスト（プレミアム） | Opus | 5cr〜 | 動的（文字数依存） |
| キャラクタートーク（通常） | Haiku | 1cr | 固定 |
| キャラクタートーク（高品質） | Sonnet | 2cr | 固定 |
| 創作ウィザード | Sonnet | 1cr | 固定 |
| エディターモード | Opus | 動的 | 動的（プロンプト長依存） |

### 3.3 動的コスト計算式

```
推定入力トークン = ceil(入力文字数 × 2)
推定出力トークン = 最大出力トークン + 思考バジェット
API原価(¥) = (入力トークン/1000 × 入力単価) + (出力トークン/1000 × 出力単価)
消費cr = max(最低cr, round(API原価 / 10 × 利益率))
```

| モデル | 入力(¥/1Kトークン) | 出力(¥/1Kトークン) | 利益率 | 最低cr |
|--------|-------------------|-------------------|--------|--------|
| Haiku | 0.12 | 0.60 | 1.6x | 0 |
| Sonnet | 0.45 | 2.25 | 1.6x | 1 |
| Sonnet+思考 | 0.45 | 2.25 | 1.6x | 5 |
| Opus | 2.25 | 11.25 | 1.4x | 30 |

### 5.3 文章アシスト — ユニットエコノミクス（動的コスト）

**Sonnet通常モード（1,000字生成の場合）:**

```
入力: ~3,000文字(6,000トークン) → ¥2.70
出力: ~4,000トークン           → ¥9.00
API原価合計                   → ¥11.70
消費cr = round(11.70 / 10 × 1.6) = 2cr
ユーザー支払: ¥19.60
PF粗利: ¥19.60 - ¥11.70 = ¥7.90 (40%)
```

**Opusモード（2,000字生成の場合）:**

```
入力: ~5,000文字(10,000トークン) → ¥22.50
出力: ~8,000トークン            → ¥90.00
API原価合計                    → ¥112.50
消費cr = max(30, round(112.50 / 10 × 1.4)) = max(30, 16) = 30cr
ユーザー支払: ¥294（Standard） / ¥264（Pro）
PF粗利: ¥294 - ¥112.50 = ¥181.50 (62%)
```

※Opusは最低30crが発動するため、短いプロンプトでも高利益率になる。

## 7. 重要な設計上の特徴

### 7.1 二重経路の収益モデル
- **レター**: JPY直接決済 → Stripe Connect即時分配（作家80%）
- **CT**: クレジット経済圏 → 月次集計振込（作家40%）

レターはStripeのdestination chargeを使い、決済と同時に作家のConnectアカウントに80%が分配される。CTはプラットフォームが一旦全額受領し、月次で40%分をtransferする。

### 7.2 クレジット消費順序による収益最適化
月次cr → 報酬cr → 有償cr の順で消費されるため、ライトユーザーの利用は作家収益を生まない。
これにより月次無料crの付与コスト（APIコスト）をプラットフォームが吸収しつつ、有償cr購入へのインセンティブを作っている。

### 7.3 Free tier ユーザーのクレジット購入が最も高利益率
Free会員のcr購入単価は¥25/crだが、作家還元は¥9.8基準の40%（¥3.92/cr）。
プラットフォームの粗利は¥21/crとなり、有料会員の4倍以上。
ただしFreeユーザーの購入率は低いため、母数での貢献は限定的。

### 7.4 0crフィーチャーの戦略的配置
コンパニオン、校正、スコアリング等のHaiku系機能を0crにすることで、クレジット消費の心理的障壁を下げ、サービスの体験価値を高めている。実際のAPIコストも極めて低い（Haiku: 1回あたり¥0.1〜0.3程度）。
