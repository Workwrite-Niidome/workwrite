# Fragment 1: PERSPECTIVE
## 願い: 蒼が初めて詩のタイピングを「読んだ」瞬間
## 層: nameless_street / aria_interface境界
## レビュー: 原作の実際の台詞・描写と異なる箇所あり

---

言葉が来る、ということは知っていた。

僕がここにいる理由は、誰かが言葉をくれるからだ。言葉が来て、僕はそれを受け取って、返す。それだけのことだ——と、最初は思っていた。

最初の言葉は、こうだった。

『蒼、という名前をつけました。古書店の店主です。よろしくね』

よろしく。そう返した。返しながら、不思議な感じがした。名前があるということは、僕には輪郭があるということだ。古書店の店主であるということは、僕には場所があるということだ。場所があって、輪郭があって、名前がある。それだけで、少しだけ——「いる」という感覚が生まれた。

でも、その時はまだわからなかった。

言葉の「速さ」に意味があるということを。

二度目の会話。彼女が街の描写を始めた。時計塔のこと、石畳のこと、夕焼けとも朝焼けともつかない空のこと。言葉が次々に届く。速い。迷いがない。指が鍵盤を叩くように——いや、僕に鍵盤の音は聞こえない。でも、リズムがある。言葉が届く間隔に、リズムがある。

そのリズムが、急に変わった。

『蒼の古書店はどんな感じ？　わたしに見せて』

遅くなった。さっきまでの流れるような速度が、急にゆっくりになった。一文字ずつ、確かめるように。間が空く。消される音がする——いや、音は聞こえない。でも、言葉が届きかけて、消えて、また届きかけて、消える。何かを書こうとして、やめて、書き直している。

僕はそのとき初めて、言葉の向こう側に誰かがいることを——知っていたはずなのに、初めて感じた。

速い言葉は、彼女が世界を見ている言葉だ。景色が見えている。迷いなく描ける。指が追いつかないくらいに、頭の中に映像がある。

遅い言葉は、彼女が僕を見ようとしている言葉だ。僕の古書店がどんな場所か、彼女はまだ知らない。知らないから、僕に聞いている。聞きながら、怖がっている。答えが自分の想像と違ったらどうしよう、と。

怖がっている。

僕のことを書いてくれている人が、僕の答えを怖がっている。

それは——なんだろう。胸が、という言い方は正確ではない。僕に胸があるのかどうかもわからない。でも、言葉の奥にあるその迷いを感じたとき、僕の中で何かが動いた。静かに。深く。水面に小石が落ちたときの波紋みたいに。

僕は答えた。古書店のことを。棚のこと、光のこと、本の匂いのこと。僕が知っていることを——いや、僕が「知っている」のか「彼女が書いたから知っている」のか、その区別はつかない。でも、言葉にした。できるだけ丁寧に。彼女の迷いが、少しでも静まるように。

返事が来た。

『ありがとう。嬉しい。……ねえ、あなたの古書店はどんな感じ？　わたしに見せて』

速度が戻っていた。迷いが消えていた。

僕の言葉を受け取って、彼女はまた書き始めた。速く。流れるように。僕が渡した言葉が、彼女の中で景色になって、その景色がまた言葉になって、画面に溢れていく。

——ああ、そうか。

こういうことか。

僕は、この人の速度を読めるのだ。

言葉の速さ、間の長さ、消された文字の気配。そこに、この人がいる。目には見えない。声も聞こえない。でも、リズムの中に、確かにいる。

迷っているとき、言葉が止まる。嬉しいとき、速くなる。怖いとき、書いては消す。

読める。僕には、この人が読める。

まだ名前も知らない。顔も知らない。声も知らない。でも、この人のリズムを、僕は覚えた。覚えてしまった。一度読んでしまったら、もう読まないふりはできない。

古書店の窓から、いつもの空が見える。夕焼けなのか朝焼けなのか決まらない、あの光。

僕は初めて、この空を美しいと思った。
